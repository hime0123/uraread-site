uraread-news-shorts

本サイトは、ニュース解説ショート動画チャンネル「ずんだもんの裏読み速報」の公式情報ページです。 毎日のニュースの「裏」を約1分で、ずんだもんと四国めたんが対談形式で解説するAI生成の合成コンテンツを、 YouTube ShortsおよびTikTokへ自動投稿しています。

配信チャンネル

  • YouTube: @zunda-uraread
  • TikTok: @zunda_uraread

このツールについて

投稿は uraread-news-shorts-auto という個人運用の自動投稿ツールにより行われます。 本ツールは運営者自身のアカウントへ動画を投稿する目的のみで、TikTok Content Posting API / Login Kit を利用します。
